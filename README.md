# AI-разработка: от нуля до агентов

Курс, который проводит от первой переменной до AI-агентов в продакшене — на одном инкрементальном проекте.

## Как начать

```bash
git clone https://github.com/hatai19/ai_builder_tutorial.git
cd tutorial

python3 -m venv venv
source venv/bin/activate      # macOS / Linux
# venv\Scripts\activate       # Windows

pip install -r requirements.txt
```

## Структура

Каждый урок — папка с теорией (`.md`), домашним заданием (`.py`) и тестами.

```
lesson 0/               Что такое программа
lesson 1 python/        Основы Python
  lesson 1.1/           Переменные
  lesson 1.2/           Типы данных и операции
  lesson 1.3/           Списки, кортежи, словари, множества
  ...
```

## Как работать

1. Прочитай теорию в файлах `.md`
2. Открой `.py` с домашкой — выполни задания
3. Запусти тесты и убедись, что всё зелёное:

```bash
pytest test_lesson_X_Y.py -v
```

4. Переходи к следующему уроку по ссылке в конце файла

## Принципы

- **Один проект** — от словаря до Kubernetes
- **Всё локально** — без платных облаков
- **Тесты с первого дня** — красные → зелёные
- **Только нужное** — без воды

# Множества (`set`)

---

**Неупорядоченное, только уникальные элементы.** Как мешок, в который нельзя положить две одинаковые штуки.

```python
tags = {"python", "fastapi", "python", "docker"}
print(tags)       # {"python", "fastapi", "docker"} — дубликат исчез
```

Фигурные скобки, но без двоеточий (в словаре — с двоеточиями).

---

## Добавление элемента

```python
tags.add("postgres")
```

---

## Проверка наличия

```python
print("python" in tags)    # True
print("java" in tags)      # False
```

Проверка `in` работает мгновенно — множеству не нужно перебирать все элементы.

---

## Зачем нужен

- Убрать дубликаты из списка
- Проверить, есть ли элемент (быстрее, чем в списке)
- Хранить теги, роли, разрешения — где важна уникальность

```python
# Убрать повторы из списка
names = ["Анна", "Борис", "Анна"]
unique = set(names)
print(unique)              # {"Анна", "Борис"}
```

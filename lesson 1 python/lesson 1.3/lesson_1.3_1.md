# Списки (`list`)

---

**Упорядоченный, изменяемый.** Элементы идут друг за другом, можно добавлять новые.

```python
fruits = ["яблоко", "банан", "киви"]
```

Квадратные скобки, элементы через запятую.

---

## Индекс — номер элемента

У каждого элемента есть порядковый номер — **индекс**. Счёт начинается с нуля:

```python
print(fruits[0])    # яблоко  (первый)
print(fruits[1])    # банан   (второй)
print(fruits[2])    # киви    (третий)
```

---

## Добавление элемента

Метод `append()` добавляет элемент в конец списка:

```python
fruits.append("груша")
print(fruits)       # ["яблоко", "банан", "киви", "груша"]
```

---

## Длина списка

Функция `len()` возвращает количество элементов:

```python
print(len(fruits))  # 4
```

---

## Зачем нужен

Список — основная коллекция для хранения набора однотипных данных: список пользователей, список товаров, список сообщений.

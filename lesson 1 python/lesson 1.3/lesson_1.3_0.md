## Зачем нужны коллекции

Переменная хранит одно значение: число, строку, `True`/`False`. А если нужно хранить *много* значений? Например, список покупок или данные пользователя с именем и возрастом?

Для этого есть **коллекции** — контейнеры, которые вмещают сразу несколько элементов.

Сегодня разберём три главных:

| Коллекция | Порядок | Можно менять | Пример |
|-----------|---------|-------------|--------|
| `list` | да | да | `["яблоко", "банан"]` |
| `tuple` | да | нет | `(10, 20)` |
| `dict` | — | да | `{"name": "Анна", "age": 25}` |
| `set` | нет | да | `{"python", "docker"}` |

Дальше — каждый тип отдельно.

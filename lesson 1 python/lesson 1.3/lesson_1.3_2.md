# Кортежи (`tuple`)

---

**Упорядоченный, НЕизменяемый.** Как список, но после создания нельзя ничего добавить или удалить.

```python
point = (10, 20)
print(point[0])     # 10
print(point[1])     # 20
```

Круглые скобки. Индексация та же — с нуля.

---

## Неизменяемость — это фича, не баг

```python
# point[0] = 99    # ОШИБКА! Кортеж нельзя изменить
```

Кортеж используют, когда данные не должны случайно поменяться: координаты на карте, RGB-цвет `(255, 128, 0)`, дни недели.

---

## Зачем нужен, если есть список

Кортеж легче и быстрее списка. Python знает, что он не изменится, и оптимизирует память. Плюс кортеж можно использовать как ключ в словаре, а список — нет.

```python
colors = {
    (255, 0, 0): "красный",
    (0, 255, 0): "зелёный",
}
```

# Словари (`dict`)

---

**Ключ → значение.** Не по номеру, а по имени (ключу). Как телефонная книга: по имени находишь номер.

```python
user = {
    "name": "Анна",
    "age": 25,
}
```

Фигурные скобки, ключ — строка, двоеточие, значение.

---

## Доступ по ключу

```python
print(user["name"])    # Анна
print(user["age"])     # 25
```

---

## Добавление нового ключа

Просто присвой значение по новому ключу — он появится:

```python
user["city"] = "Москва"
print(user)    # {"name": "Анна", "age": 25, "city": "Москва"}
```

---

## Изменение значения

По существующему ключу — перезаписывается:

```python
user["age"] = 26
```

---

## Зачем нужен

Словарь — **главная структура в бэкенде**. Когда сервер получает JSON-запрос от сайта, он превращается именно в словарь:

```json
{
  "name": "Анна",
  "email": "anna@example.com"
}
```

Жирный плюс: доступ по ключу — мгновенный. Словарю не нужно перебирать все элементы, чтобы найти нужный.

---

## Шпаргалка

```python
# list — упорядоченный, изменяемый
fruits = ["яблоко", "банан"]
fruits.append("киви")
first = fruits[0]

# tuple — упорядоченный, неизменяемый
point = (10, 20)
x = point[0]

# dict — ключ → значение
user = {"name": "Анна", "age": 25}
user["city"] = "Москва"
name = user["name"]

# set — только уникальные, без порядка
tags = {"python", "fastapi"}
tags.add("docker")
has_python = "python" in tags
```

---

## Правило большого пальца

| Ситуация | Бери |
|----------|------|
| Набор однотипных штук (фрукты, имена, числа) | `list` |
| Набор, который не должен меняться (координаты, настройки) | `tuple` |
| Данные с именами полей (пользователь, товар, запрос API) | `dict` |
| Уникальные значения, проверка «есть или нет» (теги, роли) | `set` |

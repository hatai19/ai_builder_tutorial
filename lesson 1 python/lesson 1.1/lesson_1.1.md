# Урок 1.1: Переменные

---

## Зачем нужны переменные

Представь, что ты регистрируешься на сайте. Ты вводишь имя — **«Анна»**. Чтобы сайт запомнил это имя и показал его на следующей странице, ему нужно куда-то его положить. Вот это «куда-то» — и есть **переменная**.

**Переменная** — это коробочка с именем. Ты кладёшь в неё значение и можешь использовать потом.

```
name ──→ "Анна"
```

---

## Как создать переменную

Очень просто: пишешь имя, знак `=` и значение.

```python
name = "Анна"
```

Знак `=` здесь означает **«сохрани»**, а не равенство из математики. Мы говорим Python: *«возьми значение "Анна" и сохрани его под именем name»*.

Ещё примеры:

```python
age = 25
city = "Москва"
```

Теперь в памяти компьютера есть три коробочки:

```
name ──→ "Анна"
age  ──→ 25
city ──→ "Москва"
```

## Как использовать переменную

Просто пишешь её имя — Python подставит значение:

```python
name = "Анна"
print(name)
```

Вывод:

```
Анна
```

---

## Переменную можно изменить

Коробочка не приколочена гвоздями — можно выкинуть старое значение и положить новое:

```python
name = "Анна"
print(name)       # Анна

name = "Борис"
print(name)       # Борис
```

Переменная всегда хранит **последнее**, что в неё положили.

---

## Как называть переменные

Можно использовать буквы, цифры и символ `_`. Но есть правила:

```python
user_name = "Анна"     # можно
bonus2 = 100           # можно (цифра не в начале)
_private = "скрыто"    # можно

2user = "?"            # НЕЛЬЗЯ — начинается с цифры
my-var = 10            # НЕЛЬЗЯ — дефис это минус
```

Слова разделяют **подчёркиванием** — стиль `snake_case`.

Ключевые слова Python использовать нельзя:

```python
if = 5     # НЕЛЬЗЯ — if это ключевое слово
for = 3    # НЕЛЬЗЯ — for это ключевое слово
```

---

## Итог

- **Переменная** — это имя для значения.
- Создаётся: `имя = значение`.
- Значение можно заменить в любой момент.
- Имя: буквы, цифры (не в начале), `_`. Стиль: `snake_case`.

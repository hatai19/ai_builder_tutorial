# Урок 1.2: Типы данных

---

## Зачем нужны типы

В прошлом уроке ты создавал переменные. Но значения бывают разные: число, текст, «да/нет». Python различает их — у каждого значения есть **тип**.

```python
x = 42          # число
y = 3.14        # дробное
z = "Анна"      # текст
flag = True     # да/нет
```

С числами можно считать, строки — склеивать. А число + строка? Ошибка. Поэтому важно понимать, с каким типом работаешь.

---

## `int` — целые числа

```python
age = 25
count = -3
big = 1_000_000       # подчёркивания для читаемости
```

Никаких кавычек, никаких точек.

---

## `float` — дробные числа

```python
price = 99.90
pi = 3.14
```

Всегда с точкой. `10.0` — float, `10` — int.

---

## `str` — строки (текст)

```python
name = "Анна"
city = 'Москва'      # можно одинарные или двойные кавычки
empty = ""            # пустая строка — тоже строка
```

Всё в кавычках — строка. `"42"` — строка, не число.

---

## `bool` — да / нет

```python
is_admin = True
is_blocked = False
```

Только два значения. Без кавычек, с большой буквы.

---

## Сравнения

Результат сравнения — всегда `True` или `False`:

```python
5 > 3        # True
5 < 3        # False
5 == 5       # True  (два знака равенства!)
5 != 3       # True
10 >= 10     # True
```

**Не путай:** `=` — присваивание, `==` — сравнение.

---

## Преобразование типов

Строку в число и обратно:

```python
int("42")          # 42
float("99.90")     # 99.9
str(25)            # "25"
bool(1)            # True
bool(0)            # False
bool("")           # False
```

Не всё можно: `int("Анна")` — ошибка.

---

## Итог: шпаргалка

| Тип | Пример | Что хранит |
|-----|--------|-----------|
| `int` | `42`, `-3` | Целое |
| `float` | `3.14` | Дробное |
| `str` | `"Анна"` | Текст |
| `bool` | `True`, `False` | Да/нет |

Теперь — операции с числами и строками.

# Операции со строками

---

## Склейка (конкатенация)

Знак `+` склеивает строки:

```python
"Привет, " + "Анна"        # "Привет, Анна"
"Hello" + " " + "World"    # "Hello World"
```

## Повторение

Знак `*` повторяет строку:

```python
"-" * 10     # "----------"
"ха" * 3     # "хахаха"
```

## Смешивать типы нельзя

```python
"возраст: " + 25        # ОШИБКА! Строка + число
"возраст: " + str(25)   # "возраст: 25" — сначала превратили число в строку
```

## Длина строки

`len()` — количество символов:

```python
len("Анна")     # 4
len("Hello")    # 5
len("")         # 0
```
